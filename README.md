Horizon — Smart City Explorer
A travel web app where you can search any city in the world and instantly discover hotels, restaurants and famous attractions — all powered by live APIs.
A college web development project using pure HTML, CSS and JavaScript.

What it does

Search any city
See real hotels and restaurants fetched live from Geoapify
Explore famous attractions and landmarks for that city
Click any card to see a detail page with a location map

APIs used
Unsplash
Fetches images dynamically for cities, hotels, restaurants and attractions.

Geoapify Geocoding
Converts a city name into coordinates (latitude and longitude) so we can search for places near it.

Google Maps Embed
Shows a location map on every detail page. No API key needed — uses the free embed URL.
